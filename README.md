## 🌟 About This Portfolio

> A fully custom **3D AI Engineer Portfolio** built from scratch as a single HTML file — no frameworks, no build tools, just raw creativity and code.

This portfolio features a **cinematic avatar video intro** that plays once before transitioning into a **full 3D interactive portfolio** powered by Three.js. Every detail — from the particle neural network background to the glitch-text hero to the Web Audio API sound booster — was hand-crafted to reflect the identity of an AI Engineer.

<div align="center">

```
🎬 Avatar Intro  ──►  🌀 3D Portfolio  ──►  📬 Contact
     [video]           [Three.js]           [form]
```

</div>

## ✨ Features

| Feature | Description |
|---|---|
| 🎬 **Video Intro Screen** | Custom avatar video plays once with animated circular progress ring |
| 🌌 **3D Particle Background** | Three.js neural network with 2,000 colorful particles |
| 🔊 **Web Audio Boost** | 3× audio amplifier + dynamics compressor + live boost slider |
| ✍️ **Typewriter Effect** | Animated role cycling: AI Systems → ML Pipelines → SIEM Analyzers |
| 💀 **Glitch Animation** | CSS glitch effect on hero name for cyberpunk aesthetics |
| 🖱️ **Custom Cursor** | Neon cyan dot + pink trailing ring with hover morphing |
| 🎨 **Colorful Theme** | Cyan · Hot Pink · Lime · Orange · Yellow on deep dark background |
| 📱 **Fully Responsive** | Works on mobile, tablet, and desktop |
| ♿ **Scroll Animations** | IntersectionObserver fade-in on every card and section |
| 📬 **Contact Form** | Animated send button with success feedback |
